# FocusBuddy AI Architecture

## Core Features

### Brain Dump
Capture thoughts and convert them into actionable items.

### ADHD Rescue
Provide focus assistance during overwhelm.

### Daily Briefing
Generate a summary of tasks, reminders and priorities.

### People That Matter
Track important contacts and follow-ups.

### Communication Memory
Remember interaction history and relationship context.

## Technology

Frontend: Flutter

Backend: Firebase Firestore

AI Layer: OpenAI API

## Goal

Create an AI productivity assistant that helps users stay organized, focused and accountable.
